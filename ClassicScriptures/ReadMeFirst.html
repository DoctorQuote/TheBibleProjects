Welcome!

This is free software. It is designed to run anywhere Java is available.

Our use of Java means that this software works great on Windows, Mac OS X, Linux, and elsewhere. -All we need to do to run this software is to:

  1. Make sure you have >= Java 1.6 installed.
  2. Download the content of this folder.
  3. Click on the *.jar file & let Java do the rest!

NOTE: Some might need to right-click on the above .jar file, then select the "Run With Java" option from their file manager.

Those who want to run this program from a terminal or command-line interface can do so by simply typing java -jar *.jar from within this folder.
